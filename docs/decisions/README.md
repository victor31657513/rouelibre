# Décisions d'architecture

Ce répertoire conserve les décisions d'architecture qui expliquent les choix structurants effectifs du projet et leurs conséquences.

## Nommage et statut

Chaque décision utilise le format `0001-titre-court.md`. Les statuts possibles sont : `proposé`, `accepté`, `remplacé` et `abandonné`.

Une décision acceptée ne doit pas être réécrite silencieusement. Une évolution crée une nouvelle décision qui remplace explicitement la précédente.

## Modèle

```md
# Titre

## Statut

Proposé | Accepté | Remplacé | Abandonné

## Contexte

## Décision

## Conséquences

## Alternatives considérées

## Date ou référence de PR
```
