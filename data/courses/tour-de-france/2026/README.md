# Corpus GPX du Tour de France 2026

Ce répertoire contient les parcours des 21 étapes du Tour de France 2026, provenant d'exports GPX 1.1 de VisuGPX.

Victor confirme disposer des droits de publication et d'utilisation de ce corpus. Les fichiers du répertoire `raw/` constituent des sources immuables : ils restent inchangés. Les futurs nettoyages ou transformations produisent des données séparées sans remplacer ces sources.
